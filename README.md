\# MyToken ERC20 Smart Contract



This project demonstrates the creation and deployment of a basic ERC20 token using Solidity.



\## Token Details

\- Name: MyToken

\- Symbol: MTK

\- Decimals: 18

\- Initial Supply: 1000 MTK



\## Features

\- Token transfer

\- Balance tracking

\- Allowance and approval system

\- Transfer and Approval events



\## Tools Used

\- Solidity ^0.8.0

\- Remix IDE

\- Git \& GitHub



\## Deployment

The contract was compiled and deployed using Remix IDE.

Deployment was successful and verified using transaction logs.
